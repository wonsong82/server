1. laravel new $projectName

2. attach packages
in ssh: ln -s ../../../../..Packages packages (if vagrant, use real system's path)

add this to composer.json
"repositories": [
        {
            "type": "path",
            "url": "packages/*/**",
            "options": {
                "symlink": true
            }
        }
    ],


3. set env database
4. composer require won/framework
5. install framework
php artisan framework:install
